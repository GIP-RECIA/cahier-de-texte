Liste des enseignants - synthèse des visas

Sur cet écran, seuls les enseignants s'étant connectés au moins une fois au cahier de texte apparaissent.

Les visas

2 sortes de visas peuvent être posés sur un cahier de texte :

: le visa simple indique que le cahier de texte sélectionné a été visé. Avec ce visa simple, l'enseignant pourra modifier les séances visées sans que soit conservé un historique de la séance ;
: le visa mémo indique que le cahier de texte sélectionné a été visé et que les séances du cahier de texte seront conservées dans l'état dans lequel elles étaient au moment de la pose du visa. Avec ce visa mémo, l'enseignant pourra modifier les séances visées, mais un historique de la séance au moment de la pose du visa sera conservé. Ce visa est plus consommateur de ressources qu'un visa simple.

Liste des enseignants

La liste des enseignants propose une synthèse des visas posés sur les cahiers de texte des enseignants. Le tableau comporte 6 colonnes : nom et prénom de l'enseignant, date de dernier ajout d'informations dans un cahier de texte, type (chef d'établissement ou inspection) et date du dernier visa reçu par l'enseignant, boutons des actions possibles qui concernent l'enseignant.
Le tableau est trié par ordre alphabétique des noms des enseignants, mais en cliquant sur le nom d'une colonne, le tableau est trié sur cette colonne. Un nouveau clic sur le nom de la colonne effectue le tri dans l'ordre inverse.

3 boutons sont disponibles face au nom d'un enseignant :

: permet de consulter, par semaine, le remplissage du cahier de texte d'un enseignant (les séances remplies sont colorées, les séances non remplies sont blanches avec un tour de couleur) ;
: permet de voir toutes les séances saisies par l'enseignant ;
: permet de lister les cahiers de texte de l'enseignant, ainsi que les visas qui y ont été posés.

Liste des cahiers de texte d'un enseignant

En cliquant sur le bouton associé à un enseignant, la liste des ses cahiers de texte s'affiche sur 6 colonnes : classe ou groupe, enseignement dispensé, date du dernier ajout d'information dans le cahier de texte, dernier visa posé (chef d'établissement ou inspection) sur le cahier de texte, date de ce visa, actions possibles sur le cahier de texte.
Si un cahier de texte a reçu un visa chef d'établissement et un visa inspection, alors 2 lignes sont affichées pour le cahier de texte.
Le cadenas placé à coté du visa indique qu'il s'agit d'un visa mémo.
Si une séance du cahier de texte a été modifiée après la pose d'un visa, la date de dernier ajout s'affiche en rouge dans le tableau.

Les boutons action de la liste des cahiers de texte d'un enseignant permettent :

: la pose d'un visa simple ;
: la pose d'un visa mémo ;
: le retrait du visa déjà posé ;
: l'accès aux séances du cahier de texte.

Le bouton permet de refermer la liste des cahiers de texte d'un enseignant.
Menu contextuel

Le menu contextuel propose les boutons suivants :

: enregistrer les visas posés depuis cet écran ;
: Annuler les visas posé depuis le dernier enregistrement.

Visas sur les enseignants remplaçants
Les séances effectuées dans le cadre d'un remplacement sont visées exclusivement depuis l'écran « Visa / Liste des séances par enseignant » en appliquant les critères de recherche suivants :
  • Voir les séances saisies en tant que remplaçant : « oui »
  • Enseignant remplacé : nom de l'enseignant absent qui a été remplacé par l'enseignant à viser
La recherche affiche les séances réalisées par l'enseignant remplaçant dans le cadre de son remplacement de l'enseignant sélectionné.
